# `@poppy/client-react`

Optional React renderer for Poppy.

## Status

Planned for **Phase 3**. Currently a placeholder.

## Notes

The web renderer ([`@poppy/client-web`](../client-web/)) is framework-agnostic and intended for use without React. This package is for adopters who want React-ergonomic integration: hooks, JSX hosting, ref forwarding, error boundaries, suspense for async data, etc.

## See also

- [`packages/client-web/`](../client-web/) — the framework-free renderer.
