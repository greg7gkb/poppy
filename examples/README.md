# Examples

Sample applications using Poppy.

## Status

Examples are added per-phase as the corresponding renderers are implemented.

## Layout (when populated)

- `examples/server-node/` — a small Node.js server that emits Poppy documents.
- `examples/web/` — minimal HTML page that renders a document via `@poppy/client-web`.
- `examples/android/` — minimal Compose app.
- `examples/ios/` — minimal SwiftUI app.

Each example is a self-contained, runnable project with its own README and build instructions.
