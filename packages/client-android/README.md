# `poppy-android`

Kotlin + Jetpack Compose renderer for Poppy.

## Status

Planned for **Phase 2**. Currently a placeholder.

## Build system

This package is a Gradle module (Kotlin DSL). It uses native Android tooling — no Nx/Turborepo wrapping. See [ADR-0002](../../docs/adr/0002-monorepo-structure.md).
