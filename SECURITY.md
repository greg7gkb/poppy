# Security Policy

## Supported versions

Poppy is in pre-v0.1 alpha. There are no supported versions yet. Once we cut a stable release, this section will list which release lines receive fixes.

## Reporting an issue

If you believe you've found a security issue in Poppy, please **do not open a public GitHub issue**. Instead:

1. Open a [private security advisory](https://github.com/greg7gkb/poppy/security/advisories/new) on GitHub, or
2. Email the maintainer at the address listed on recent commits in `git log`.

Please include:

- A description of the issue and its impact.
- Steps to reproduce, or a minimal proof-of-concept.
- Any suggested mitigations.

You should receive an acknowledgement within 7 days. We will work with you on a coordinated disclosure timeline before any public discussion.
